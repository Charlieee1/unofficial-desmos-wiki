Desmos Documentation (UNOFFICIAL)

What is this site?

Desmos Graphing Calculator is a web-based graphing calculator packed with interesting features that allow users to make all sorts of fascinating things. However, Desmos's official documentation— while sufficient for most users— does not completely do Desmos justice, as it does not cover many of the tool's more obscure features and techniques. This site was created to remedy that, providing a deep dive into many Desmos features and techniques of which users might be unaware.
